# My Landing Page

Built with Next.js and Tailwind CSS.

## Run locally
1. npm install
2. npm run dev
3. open http://localhost:3000

## Vercel v0 usage
Hero block was generated using Vercel v0 and adapted into `app/components/HeroFromV0.jsx`. Other components (Features, Testimonials, Contact, Footer) were hand-coded using Tailwind CSS.

## Live
Deployed at: https://my-landing-six.vercel.app/
